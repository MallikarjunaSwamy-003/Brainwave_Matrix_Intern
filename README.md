# 🏧 Python ATM Interface
A simple and secure console-based ATM interface built using Python.
## 🚀 Features
- ✅ PIN authentication
- 💰 Check balance
- 📥 Deposit money
- 💸 Withdraw money
- 🧾 View transaction history
- 🔐 3 attempts for PIN entry
- 🛑 Secure exit after failed attempts

## 🛠️ Tech Stack
- Python 3
- Console UI
- `getpass` for PIN security
